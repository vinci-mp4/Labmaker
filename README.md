## 🏭 MakerLab - Sistema de Gerenciamento de Laboratório
Sistema web para gerenciamento de um laboratório maker, permitindo o cadastro de usuários internos e visitantes, agendamento de uso de máquinas e controle de matéria-prima.

✨ Funcionalidades
👥 Sistema de Usuários
Login para Visitantes: Solicitação de serviços

Login Administrativo: Aprovação de requisições e gestão do sistema

Cadastro de novos usuários (interno e externo)

🛠️ Sistema de Agendamento
Seleção de máquinas disponíveis (impressoras 3D, corte a laser, fresadoras, etc.)

Definição do serviço a ser realizado

Estimativa de tempo de uso das máquinas

Escolha de matéria-prima compatível com cada máquina

Estimativa de quantidade de material

Sistema de bloqueio por indisponibilidade ou limites atingidos

📊 Painel Administrativo
Aprovação/recusa de requisições

Controle de estoque de matéria-prima

Visualização de agendamentos

Gerenciamento de máquinas e materiais

🗃️ Banco de Dados
Cadastro de máquinas e suas especificações

Controle de matéria-prima e insumos

Registro de usuários e permissões

Histórico de requisições e serviços

🚀 Tecnologias Previstas
Frontend: HTML, CSS, JavaScript (React/Angular/Vue ou similar)

Backend: Node.js/Python/Java (a definir)

Banco de Dados: PostgreSQL/MySQL

Autenticação: Sistema de login com diferentes níveis de acesso
